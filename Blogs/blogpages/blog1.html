Most three Visual Studio Code Extensions

23 Dec 2016

Recently I’ve run into a few simple things that are missing from Visual Studio Code out of the box. Here are the plugins I think everyone needs. 1. toggle-upper-case I spend most of my time in full visual studio and use the case switching shortcut all the time. I was surprised that VS Code doesn’t have this shortcut built-in. The toggle-upper-case plugin does a great job of flipping between UPPERCASE and camelCase by hitting CTRL+U on the highlighted text.

2. Sort for Visual Studio Code Often I like to sort parameters or other lines of code, another thing that’s missing a shortcut in VS Code. The Sort for Visual Studio Code plugin is great. Highlight some lines of code and press SHIFT+ALT+S and it’s sorted, press again to reverse.

3. Todo Tree It’s great to keep track on TODOs in code. I’ve mentioned before about the use of the Better Comments plugin, which is great when you’re viewing a single file, but if you want to see all your TODOs in a list, then check out the Todo Tree Visual Studio Code plugin. This extension quickly searches your workspace for comment tags like TODO and FIXME, and displays them in a tree view in the explorer pane.
